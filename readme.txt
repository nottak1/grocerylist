GROCERY LIST PROJECT

General Info :
- Read and write to firestore, a non-sql database
- uses firebase authentication with a collection of users in firestore
- Customized user settings and lists after signup/login
- Use of bootstrap

Technologies:
- Microsoft azure hosting
- Firebase Firestore database
- HTML, CSS
- JavaScript
- Bootstrap

Content:
Top level of project folder:
    - .gitignore        # Git ignore file
    - index.html        # Page on first entry
    - login.html        # Page to login
    - main.html         # Main page after login
    - storelist.html    # List of all the storelist

    It has the following subfolders:
    - CSS               # Main folder for all css styling
    - /styling.css      # All external styling
    - images            # Folder for all images
    - scripts           # Folder for all scripts
    - /scripts.js       # Main functions of app
    - /firebase_api_team19.js   # Firebase api
